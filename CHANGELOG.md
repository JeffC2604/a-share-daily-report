# Changelog

## v0.4.1 - 2026-05-26

### Changed

- 增强云端板块数据源兼容性，板块接口请求增加更完整 headers。
- 板块接口失败日志增加请求 URL、HTTP 状态码、响应前 300 字符和 headers 生效情况。
- 东方财富板块主接口失败后自动尝试备用板块接口。
- 云端仍无法获取板块数据时，日报明确提示“云端环境下板块接口暂时不可用”。

### Safety

- 不改变数据抓取主流程。
- 不改变候选池筛选逻辑。
- 不增加个股推荐。
- 不增加投资建议。
- 保留 `risk_rules.py` 禁用词扫描。

## v0.1.0 - 2026-05-25

第一阶段完成。

### Added

- `config.json` 配置文件，支持配置报告日期、基金代码、指数展示列表和输出目录。
- 指数数据抓取。
- 基金估值抓取。
- 东方财富主指数接口失败后自动切换备用指数接口。
- 本地 `fallback_data.json` 备用数据结构。
- Markdown 日报输出。
- HTML 日报输出，包含基础表格样式和数据源状态高亮。
- 轻量市场观察：数据完整性、指数整体状态、基金估值波动和置信度。
- 禁用词安全扫描，阻止生成包含高风险表述的日报。

### Notes

- 不包含板块分析。
- 不包含个股推荐。
- 不包含投资建议。
- 数据源可能延迟、失败或返回异常，需查看日报中的数据源状态。
